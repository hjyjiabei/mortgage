# 房贷计算器概要设计文档

## 1. 项目概述

### 1.1 文档信息

| 项目 | 说明 |
|------|------|
| 项目名称 | 房贷计算器 |
| 文档版本 | V1.0 |
| 编写日期 | 2026-04-13 |
| 技术架构 | 前后端分离 |
| 后端技术 | JDK 21 + Spring Boot 3 + Maven + MySQL 8 |
| 前端技术 | Vue 2 + Element UI + ECharts |

### 1.2 项目背景

本项目旨在开发一款功能完善的房贷计算器，支持等额本息、等额本金两种主流还款方式的计算，提供还款明细展示、方案对比、提前还款模拟、PDF 导出等功能，帮助用户做出更优的贷款决策。

### 1.3 系统目标

1. 提供准确的房贷计算功能
2. 支持多种还款方案对比
3. 支持提前还款模拟
4. 提供可视化图表展示
5. 支持还款计划导出

### 2.3 技术架构说明

#### 2.3.1 分层架构

系统采用经典的分层架构，各层职责明确：

| 层级 | 职责 | 关键类 |
|------|------|--------|
| Controller 层 | 接收请求、参数校验、返回响应 | XxxController |
| Service 层 | 业务逻辑处理、事务管理 | XxxService |
| Mapper 层 | 数据访问、CRUD 操作（MyBatis-Plus） | XxxMapper |
| Model 层 | 实体类、DTO、VO | Entity、DTO、VO |

#### 2.3.2 前后端分离

- **前端**：独立部署，负责页面渲染和用户交互
- **后端**：提供 RESTful API，负责业务处理和数据存储
- **通信**：通过 HTTP/HTTPS 协议，数据格式为 JSON

#### 2.3.3 RESTful API 设计

| HTTP 方法 | 用途 | 示例 |
|-----------|------|------|
| GET | 查询资源 | GET /api/plan/{id} |

| PUT | 更新资源 | PUT /api/plan/{id} |
| DELETE | 删除资源 | DELETE /api/plan/{id} |

#### 2.3.4 MyBatis-Plus 配置说明

**核心配置**：

```java
// MybatisPlusConfig.java
@Configuration
@MapperScan("com.mortgage.mapper")
public class MybatisPlusConfig {
    
    // 分页插件
    @Bean
    public MybatisPlusInterceptor mybatisPlusInterceptor() {
        MybatisPlusInterceptor interceptor = new MybatisPlusInterceptor();
        interceptor.addInnerInterceptor(new PaginationInnerInterceptor(DbType.MYSQL));
        return interceptor;
    }
    
    // 逻辑删除配置（可选）
    // 全局配置在 application.yml 中设置
}
```

**application.yml 配置**：

```yaml
mybatis-plus:
  mapper-locations: classpath:mapper/*.xml
  type-aliases-package: com.mortgage.model.entity
  configuration:
    map-underscore-to-camel-case: true
    log-impl: org.apache.ibatis.logging.stdout.StdOutImpl
  global-config:
    db-config:
      id-type: auto
      logic-delete-field: deleted
      logic-delete-value: 1
      logic-not-delete-value: 0
```

**Mapper 接口规范**：

```java
// 继承 BaseMapper 获得基础 CRUD 方法
public interface LoanRequestMapper extends BaseMapper<LoanRequest> {
    // 自定义查询方法
    @Select("SELECT * FROM loan_request WHERE user_id = #{userId}")
    List<LoanRequest> selectByUserId(@Param("userId") String userId);
}

public interface RepaymentDetailMapper extends BaseMapper<RepaymentDetail> {
    // 批量插入
    void insertBatch(@Param("list") List<RepaymentDetail> list);
    
    // 按计划ID查询
    List<RepaymentDetail> selectByPlanId(@Param("planId") Long planId);
}
```

**常用方法对照**：

| JPA 方法 | MyBatis-Plus 方法 | 说明 |
|----------|-------------------|------|
| save() | insert() | 新增记录 |
| saveAll() | insertBatch() | 批量新增（需自定义） |
| findById() | selectById() | 按ID查询 |
| findAll() | selectList() | 查询全部 |
| deleteById() | deleteById() | 按ID删除 |
| getOne() | selectOne() | 查询单条 |

#### 3.1.1 模块职责说明

| 模块名称 | 职责 | 核心功能 |
|----------|------|----------|
| 计算模块 (Calculate) | 执行房贷核心计算逻辑 | 等额本息计算、等额本金计算 |
| 对比模块 (Compare) | 多方案对比分析 | 双方案对比、差异计算 |
| 提前还款模块 (Prepay) | 提前还款模拟计算 | 缩期计算、减月供计算、对比 |
| 计划管理模块 (Plan) | 还款计划存储与查询 | 保存计划、查询明细、历史记录 |
| 导出模块 (Export) | 数据导出功能 | PDF 导出、格式化输出 |
| 公共模块 (Common) | 公共功能支撑 | 参数校验、异常处理、日志记录 |

### 4.3 业务规则

#### 4.3.1 计算规则

##### 等额本息计算规则

```
输入参数：
  - loanAmount: 贷款本金（元）
  - loanTerm: 贷款期限（月）
  - annualRate: 年利率（小数形式，如 0.043）

计算步骤：
  1. 计算月利率
     monthlyRate = annualRate / 12
  
  2. 计算月供（固定）
     monthlyPayment = loanAmount × monthlyRate × (1 + monthlyRate)^loanTerm 
                    / [(1 + monthlyRate)^loanTerm - 1]
  
  3. 遍历计算每期明细
     for period = 1 to loanTerm:
       - 当期利息 = 剩余本金 × monthlyRate
       - 当期本金 = monthlyPayment - 当期利息
       - 剩余本金 = 上期剩余本金 - 当期本金
       - 累计还款 += monthlyPayment
  
  4. 汇总结果
     - totalPayment = monthlyPayment × loanTerm
     - totalInterest = totalPayment - loanAmount
     - interestRatio = totalInterest / loanAmount × 100%

精度要求：
  - 所有金额精确到分（小数点后2位）
  - 利率精确到小数点后6位
```

##### 等额本金计算规则

```
输入参数：
  - loanAmount: 贷款本金（元）
  - loanTerm: 贷款期限（月）
  - annualRate: 年利率（小数形式）

计算步骤：
  1. 计算月利率
     monthlyRate = annualRate / 12
  
  2. 计算每月固定本金
     monthlyPrincipal = loanAmount / loanTerm
  
  3. 计算月供递减额
     monthlyDecrease = monthlyPrincipal × monthlyRate
  
  4. 遍历计算每期明细
     remainingPrincipal = loanAmount
     for period = 1 to loanTerm:
       - 当期本金 = monthlyPrincipal
       - 当期利息 = remainingPrincipal × monthlyRate
       - 当期月供 = 当期本金 + 当期利息
       - 剩余本金 = remainingPrincipal - 当期本金
       - 累计还款 += 当期月供
  
  5. 汇总结果
     - firstPayment = 第1期月供
     - lastPayment = 第loanTerm期月供
     - totalPayment = 累计还款总和
     - totalInterest = totalPayment - loanAmount

特点：
  - 月供逐月递减，递减额固定
  - 前期压力大，总利息少
```

#### 4.3.2 提前还款计算规则

##### 缩期方式计算规则

```
输入参数：
  - originalPlan: 原还款计划
  - prepayPeriod: 提前还款期数
  - prepayAmount: 提前还款金额

计算步骤：
  1. 获取提前还款时的剩余本金
     remainingPrincipal = 原计划第prepayPeriod期的剩余本金
  
  2. 计算新剩余本金
     newRemainingPrincipal = remainingPrincipal - prepayAmount
  
  3. 计算新月供（保持基本不变）
     newMonthlyPayment ≈ 原月供（或微调）
  
  4. 计算新期限（反推）
     使用等额本息/等额本金公式，以新剩余本金和新月供反推期限
     newTerm = 通过公式计算得出
  
  5. 生成新还款明细
     以新剩余本金、新期限重新计算
  
  6. 计算节省利息
     savedInterest = 原计划剩余利息 - 新计划总利息

限制条件：
  - 提前还款金额 ≤ 剩余本金
  - 新期限 ≥ 1（不能为0）
```

##### 减月供方式计算规则

```
输入参数：
  - originalPlan: 原还款计划
  - prepayPeriod: 提前还款期数
  - prepayAmount: 提前还款金额

计算步骤：
  1. 获取提前还款时的剩余本金和剩余期限
     remainingPrincipal = 原计划第prepayPeriod期的剩余本金
     remainingTerm = 原期限 - prepayPeriod
  
  2. 计算新剩余本金
     newRemainingPrincipal = remainingPrincipal - prepayAmount
  
  3. 保持期限不变
     newTerm = remainingTerm
  
  4. 计算新月供
     使用原还款方式公式，以新剩余本金和原剩余期限计算
     newMonthlyPayment = 通过公式计算得出
  
  5. 生成新还款明细
  
  6. 计算节省利息
     savedInterest = 原计划剩余利息 - 新计划总利息

特点：
  - 月供降低，压力减轻
  - 节省利息相对较少
```

| 参数 | 校验规则 | 错误码 | 错误提示 |
|------|----------|--------|----------|
| loanAmount | 1万 ≤ amount ≤ 1亿 | E001 | 贷款金额应在1万-1亿之间 |
| loanTerm | 1 ≤ term ≤ 360 | E002 | 贷款期限应在1-360个月之间 |
| annualRate | 0.0001 ≤ rate ≤ 0.2 | E003 | 年利率应在0.01%-20%之间 |

| prepayPeriod | > 已还款期数 | E006 | 提前还款期数应在已还款期数之后 |
| repaymentMethod | 有效枚举值 | E007 | 还款方式不正确 |

### 5.4 数据存储策略

| 场景 | 存储策略 | 说明 |
|------|----------|------|
| 实时计算（不保存） | 不存储 | 前端传参 → 后端计算 → 直接返回结果 |
| 需要导出 PDF | 存储完整数据 | 存储 loan_request、repayment_plan、repayment_detail |
| 方案对比 | 存储两个方案 | 同时存储两个计划的完整数据，记录 compare_record |
| 提前还款模拟 | 存储新计划 | 原计划标记作废，新计划关联原计划，记录 prepay_record |
| 历史查询 | 读取数据库 | 根据 user_id 或 plan_no 查询历史计算记录 |

```sql
-- ============================================================
-- 1. 贷款申请记录表
-- ============================================================
DROP TABLE IF EXISTS loan_request;
CREATE TABLE loan_request (
    id BIGINT PRIMARY KEY AUTO_INCREMENT COMMENT '主键ID',
    loan_amount DECIMAL(12, 2) NOT NULL COMMENT '贷款金额（元）',
    loan_term INT NOT NULL COMMENT '贷款期限（月）',
    annual_rate DECIMAL(8, 6) NOT NULL COMMENT '年利率（小数形式，如0.043000）',

-- ============================================================
-- 2. 还款计划表
-- ============================================================
DROP TABLE IF EXISTS repayment_plan;
CREATE TABLE repayment_plan (
    id BIGINT PRIMARY KEY AUTO_INCREMENT COMMENT '主键ID',

    loan_request_id BIGINT NOT NULL COMMENT '关联贷款申请ID',
    loan_amount DECIMAL(12, 2) NOT NULL COMMENT '贷款本金（元）',
    loan_term INT NOT NULL COMMENT '原始贷款期限（月）',
    actual_term INT NOT NULL COMMENT '实际还款期限（月，提前还款后可能变化）',
    annual_rate DECIMAL(8, 6) NOT NULL COMMENT '年利率',
    monthly_rate DECIMAL(10, 8) NOT NULL COMMENT '月利率',
    repayment_method VARCHAR(30) NOT NULL COMMENT '还款方式',
    total_payment DECIMAL(14, 2) NOT NULL COMMENT '总还款额（元）',
    total_interest DECIMAL(14, 2) NOT NULL COMMENT '总利息（元）',
    total_principal DECIMAL(14, 2) NOT NULL COMMENT '总本金（元）',

    last_payment_date DATE NOT NULL COMMENT '最后还款日期',
    status TINYINT NOT NULL DEFAULT 1 COMMENT '状态: 0-已作废, 1-有效, 2-已结清',
    source VARCHAR(20) DEFAULT 'CALCULATION' COMMENT '来源: CALCULATION(新计算), PREPAY(提前还款产生)',
    parent_plan_id BIGINT DEFAULT NULL COMMENT '父计划ID（提前还款场景）',
    created_at DATETIME NOT NULL DEFAULT CURRENT_TIMESTAMP COMMENT '创建时间',
    updated_at DATETIME NOT NULL DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP COMMENT '更新时间',

    INDEX idx_plan_no (plan_no),
    INDEX idx_status (status),

    FOREIGN KEY (loan_request_id) REFERENCES loan_request(id) ON DELETE CASCADE

-- ============================================================
-- 3. 还款明细表
-- ============================================================
DROP TABLE IF EXISTS repayment_detail;
```

### 6.5 视图创建

```sql
-- ============================================================
-- 8. 创建视图 - 常用查询优化
-- ============================================================
-- 还款计划汇总视图
DROP VIEW IF EXISTS v_plan_summary;
CREATE VIEW v_plan_summary AS
SELECT 
    rp.id,
    rp.plan_no,
    lr.loan_amount / 10000 AS loan_amount_wan,
    lr.loan_term,
    lr.annual_rate * 100 AS annual_rate_percent,
    CASE rp.repayment_method 
        WHEN 'EQUAL_PRINCIPAL_INTEREST' THEN '等额本息'
        WHEN 'EQUAL_PRINCIPAL' THEN '等额本金'
    END AS repayment_method_name,
    rp.total_payment,
    rp.total_interest,
    rp.total_principal,
    rp.first_payment_date,
    rp.last_payment_date,
    rp.status,
    rp.created_at
FROM repayment_plan rp
JOIN loan_request lr ON rp.loan_request_id = lr.id
WHERE rp.status = 1;

-- 提前还款记录汇总视图
DROP VIEW IF EXISTS v_prepay_summary;
CREATE VIEW v_prepay_summary AS
SELECT 
    pr.id,
    pr.prepay_amount,
    pr.prepay_date,
    CASE pr.prepay_type 
        WHEN 'SHORTEN_TERM' THEN '缩期'
        WHEN 'REDUCE_PAYMENT' THEN '减月供'
    END AS prepay_type_name,
    pr.saved_interest,
    rp.plan_no AS original_plan_no,
    rp_new.plan_no AS new_plan_no,
    pr.created_at
FROM prepay_record pr
JOIN repayment_plan rp ON pr.original_plan_id = rp.id
LEFT JOIN repayment_plan rp_new ON pr.new_plan_id = rp_new.id;
```

### 6.6 存储过程

```sql
-- ============================================================
-- 9. 存储过程 - 计算等额本息月供（可选，服务端计算更推荐）
-- ============================================================
DROP PROCEDURE IF EXISTS sp_calc_monthly_payment;
DELIMITER //
CREATE PROCEDURE sp_calc_monthly_payment(
    IN p_loan_amount DECIMAL(12, 2),
    IN p_loan_term INT,
    IN p_annual_rate DECIMAL(8, 6),
    IN p_method VARCHAR(30),
    OUT p_monthly_payment DECIMAL(12, 2),
    OUT p_total_interest DECIMAL(14, 2)
)
BEGIN
    DECLARE v_monthly_rate DECIMAL(10, 8);
    DECLARE v_factor DECIMAL(20, 10);
    DECLARE v_total_payment DECIMAL(14, 2);
    
    SET v_monthly_rate = p_annual_rate / 12;
    
    IF p_method = 'EQUAL_PRINCIPAL_INTEREST' THEN
        -- 等额本息公式
        SET v_factor = POWER(1 + v_monthly_rate, p_loan_term);
        SET p_monthly_payment = ROUND(p_loan_amount * v_monthly_rate * v_factor / (v_factor - 1), 2);
        SET v_total_payment = p_monthly_payment * p_loan_term;
        SET p_total_interest = v_total_payment - p_loan_amount;
    ELSE
        -- 等额本金（首月月供）
        SET p_monthly_payment = ROUND(p_loan_amount / p_loan_term + p_loan_amount * v_monthly_rate, 2);
        -- 等额本金总利息
        SET p_total_interest = ROUND(p_loan_amount * v_monthly_rate * (p_loan_term + 1) / 2, 2);
    END IF;
END //
DELIMITER ;
```

3. 执行 `npm install && npm run build`
