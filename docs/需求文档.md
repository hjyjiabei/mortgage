# 房贷计算器需求文档

## 1. 项目概述

### 1.1 项目背景

购房是大多数家庭的重要决策，房贷作为购房的主要融资方式，其还款方式、期限、利率等因素直接影响借款人的财务规划。目前市面上的房贷计算工具功能单一，缺乏全面的还款方案对比和提前还款模拟功能，无法满足用户多元化的决策需求。

本项目旨在开发一款功能完善、操作便捷的房贷计算器，帮助用户清晰了解不同还款方式的差异，提前还款的影响，从而做出更优的财务决策。

### 1.2 项目目标

1. 提供准确的房贷计算功能，支持等额本息、等额本金两种主流还款方式
2. 支持还款明细查询，展示每期还款的本金、利息、余额等信息
3. 支持多种还款方案对比，帮助用户选择最优方案
4. 支持提前还款模拟，计算提前还款后的新还款计划
5. 支持还款计划导出为 PDF 格式，方便用户保存和打印
6. 提供可视化图表展示，直观呈现还款构成和趋势

### 1.3 项目范围

| 范围项 | 说明 |
|--------|------|
| 用户群体 | 有购房贷款需求的个人用户 |
| 贷款类型 | 商业贷款（预留公积金贷款、组合贷款扩展） |
| 还款方式 | 等额本息、等额本金 |
| 平台支持 | Web 浏览器（PC端、移动端响应式） |

### 2.2 功能详细说明

#### 2.2.1 F01 等额本息计算

**功能描述**：用户输入贷款金额、期限、利率，系统计算等额本息还款计划。

**输入参数**：
| 参数 | 类型 | 必填 | 校验规则 |
|------|------|------|----------|
| 贷款金额 | Decimal | 是 | 1-10000万元，精确到小数点后2位 |
| 贷款期限 | Integer | 是 | 1-360月（最长30年） |
| 年利率 | Decimal | 是 | 0.01%-20%，精确到小数点后4位 |
| 放款日期 | Date | 否 | 默认当月 |

**输出结果**：
| 字段 | 说明 |
|------|------|
| 月供金额 | 每月固定还款额 |
| 总还款额 | 本金+利息总和 |
| 总利息 | 所有利息之和 |
| 还款明细列表 | 每期本金、利息、余额等 |

**计算公式**：
```
月供 = [贷款本金 × 月利率 × (1 + 月利率)^还款月数] / [(1 + 月利率)^还款月数 - 1]
每月利息 = 剩余本金 × 月利率
每月本金 = 月供 - 每月利息
```

#### 2.2.2 F02 等额本金计算

**功能描述**：用户输入贷款金额、期限、利率，系统计算等额本金还款计划。

**输入参数**：同 F01

**输出结果**：
| 字段 | 说明 |
|------|------|
| 首月还款 | 第一个月还款金额 |
| 末月还款 | 最后一个月还款金额 |
| 月供递减额 | 每月递减金额 |
| 总还款额 | 本金+利息总和 |
| 总利息 | 所有利息之和 |
| 还款明细列表 | 每期本金、利息、余额等 |

**计算公式**：
```
每月本金 = 贷款本金 / 还款月数
每月利息 = 剩余本金 × 月利率
每月还款 = 每月本金 + 每月利息
月供递减额 = 每月本金 × 月利率
```

#### 2.2.3 F03 还款明细展示

**功能描述**：以表格形式展示每期还款的详细信息。

**明细字段**：
| 字段 | 说明 |
|------|------|
| 期数 | 第几个月（1-N） |
| 还款日期 | 当期还款日期 |
| 月供金额 | 当期还款总额 |
| 本金部分 | 当期还款中的本金 |
| 利息部分 | 当期还款中的利息 |
| 剩余本金 | 还款后的剩余本金 |
| 累计还款 | 截止当期的累计还款额 |

**交互要求**：
- 支持分页展示
- 支持按期数搜索
- 支持表格排序

#### 2.2.4 F04 图表展示

**功能描述**：使用图表可视化展示还款数据。

**图表类型**：
| 图表 | 说明 |
|------|------|
| 饼图 | 总体本金/利息占比 |
| 折线图 | 每月本金利息变化趋势 |
| 堆叠柱状图 | 每月还款构成 |
| 面积图 | 累计还款、累计本金、累计利息趋势 |

#### 2.2.5 F05 方案对比

**功能描述**：同时展示两种还款方案的对比结果。

**对比维度**：
| 维度 | 说明 |
|------|------|
| 总利息差额 | 两种方案的总利息差 |
| 首月月供差额 | 首月还款金额差 |
| 总还款差额 | 总还款金额差 |
| 利息占比对比 | 利息占本金比例对比 |

**展示形式**：
- 并排表格对比
- 差异高亮显示
- 对比图表

#### 2.2.6 F06 提前还款模拟

**功能描述**：模拟提前还款后的新还款计划。

**输入参数**：
| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| 原还款计划 | Object | 是 | 已有的还款计划 |
| 提前还款金额 | Decimal | 是 | 不少于1万元 |
| 提前还款期数 | Integer | 是 | 在第几期提前还款 |
| 处理方式 | Enum | 是 | 缩期/减月供 |

**输出结果**：
| 字段 | 说明 |
|------|------|
| 新还款计划 | 调整后的完整还款计划 |
| 节省利息 | 相比原计划节省的利息 |
| 新期限/新月供 | 根据处理方式变化的数据 |

**处理方式说明**：
| 方式 | 说明 |
|------|------|
| 缩期 | 保持月供基本不变，缩短还款期限 |
| 减月供 | 保持还款期限不变，降低月供金额 |

#### 2.2.7 F07 提前还款方式对比

**功能描述**：对比缩期和减月供两种提前还款方式的效果。

**对比内容**：
| 维度 | 缩期 | 减月供 |
|------|------|--------|
| 新还款期限 | 缩短 | 不变 |
| 新月供金额 | 基本不变 | 减少 |
| 节省利息 | 较多 | 较少 |
| 月供压力 | 不变 | 降低 |

#### 2.2.8 F08 PDF 导出

**功能描述**：将还款计划导出为 PDF 文件。

**导出内容**：
- 贷款基本信息
- 还款汇总信息
- 还款明细表格
- 图表（可选）

**导出格式**：
- 文件命名：还款计划_YYYYMMDD_HHmmss.pdf
- A4 纸张，支持分页打印

## 6. 业务对象分析

### 6.1 核心业务对象

#### LoanRequest（贷款申请）

| 字段 | 类型 | 说明 | 校验规则 |
|------|------|------|----------|
| loanAmount | BigDecimal | 贷款金额（元） | > 0，精确到分 |
| loanTerm | Integer | 贷款期限（月） | 1-360 |
| annualRate | BigDecimal | 年利率 | 0.0001-0.2 |
| repaymentMethod | Enum | 还款方式 | EQUAL_PRINCIPAL_INTEREST / EQUAL_PRINCIPAL |
| startDate | LocalDate | 放款日期 | 有效日期 |
| loanType | Enum | 贷款类型 | COMMERCIAL / PROVIDENT_FUND / COMBINED |

#### RepaymentPlan（还款计划）

| 字段 | 类型 | 说明 |
|------|------|------|
| planId | String | 计划唯一标识 |
| requestInfo | LoanRequest | 关联的贷款申请 |
| totalPayment | BigDecimal | 总还款额 |
| totalInterest | BigDecimal | 总利息 |
| totalPrincipal | BigDecimal | 总本金 |
| details | List\<RepaymentDetail\> | 还款明细列表 |

#### RepaymentDetail（还款明细）

| 字段 | 类型 | 说明 |
|------|------|------|
| period | Integer | 期数 |
| paymentDate | LocalDate | 还款日期 |
| monthlyPayment | BigDecimal | 月供金额 |
| principal | BigDecimal | 本金部分 |
| interest | BigDecimal | 利息部分 |
| remainingPrincipal | BigDecimal | 剩余本金 |
| cumulativePayment | BigDecimal | 累计还款 |

#### PrepayRequest（提前还款请求）

| 字段 | 类型 | 说明 |
|------|------|------|
| originalPlanId | String | 原计划ID |
| prepayAmount | BigDecimal | 提前还款金额 |
| prepayPeriod | Integer | 提前还款期数 |
| prepayType | Enum | 处理方式 |

#### PrepayResult（提前还款结果）

| 字段 | 类型 | 说明 |
|------|------|------|
| newPlan | RepaymentPlan | 新还款计划 |
| savedInterest | BigDecimal | 节省利息 |
| newTerm | Integer | 新期限（缩期时） |
| newPayment | BigDecimal | 新月供（减月供时） |

### 6.2 枚举定义

```java
// 还款方式
public enum RepaymentMethod {
    EQUAL_PRINCIPAL_INTEREST,  // 等额本息
    EQUAL_PRINCIPAL            // 等额本金
}

// 提前还款类型
public enum PrepayType {
    SHORTEN_TERM,      // 缩短期限
    REDUCE_PAYMENT     // 减少月供
}

// 贷款类型
public enum LoanType {
    COMMERCIAL,        // 商业贷款
    PROVIDENT_FUND,    // 公积金贷款
    COMBINED           // 组合贷款
}
```
